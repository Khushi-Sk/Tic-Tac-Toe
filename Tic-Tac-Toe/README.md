# Tic Tac Toe in JavaScript

1. We start the game.
2. There are 2 players X & O. [rule]
3. We store their name.
4. X plays first. [rule]
5. They mark their spot:

    X _ _ 
    _ _ _
    _ _ _

6. O, plays:

    X _ _ 
    _ O _
    _ _ _

 7. Game continues, until one of these happens:
    1. X wins
    2. O wins
    3. Draw
    4. More moves left

## Emerging Requirements
